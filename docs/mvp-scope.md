# VANTAGE — MVP Scope

> This document is in progress. Full scope documentation coming soon.

## What MVP Delivers

- A structured inventory of AI tools in use across the organization
- Pre-populated library of common AI tools plus manual entry option
- Ownership and authorization tracking per tool
- Data exposure classification per tool
- Lightweight risk signal (Low / Medium / High) per tool

## What MVP Does Not Include

- Network scanning or automated AI detection
- Browser extension monitoring
- Multi-tenant SaaS
- Full compliance reporting
- Authentication / user accounts (Phase 2)

VANTAGE MVP is structured visibility through intentional human input.
Scope expands in Phase 2 and beyond.
