# VANTAGE — Architecture

> This document is in progress. Full architecture documentation coming soon.

## Overview

VANTAGE is built on a three-layer pattern:

1. **Discovery** — identifying AI tools in use across the organization
2. **Ownership** — establishing who is responsible for each tool and whether it is Authorized or Unauthorized
3. **Data Exposure** — classifying what kind of data each tool touches

Full architecture detail, diagrams, and design decisions will be documented here as the build progresses.
