# VANTAGE — Roadmap

> This document is in progress. Full roadmap coming soon.

## Phase 1 — MVP

- Core tool inventory (identity, ownership, data exposure, risk signal)
- Pre-populated library of common AI tools
- Manual entry for unlisted tools
- Basic dashboard view

## Phase 2

- Automated detection (network-level shadow AI discovery)
- Multi-user support
- Export for audit and compliance reporting

## Phase 3

- Multi-tenant SaaS option
- Compliance framework mapping (EU AI Act, ISO 42001)
- API integrations with existing security tooling

Roadmap is subject to change as the project evolves.
